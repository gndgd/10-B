Ctrl-A : tüm dosyayı (yazıyı) seçer tümünü seç anlamına gelmekte
Ctrl-S : Kaydetme işlemi yapar 
Ctrl-Z : Geri al (yapılan komutu geri alır)
Ctrl-Y : Yinele (Geri alma işlemini geri al)
Ctrl-W : Sekmeyi kapatır 
Ctrl-P : Yazdır
Ctrl-O : Aç (kaydedilen dosyayı açmaya yarar)
Ctrl-C : Kopyalar 
Ctrl-X : Keser
Ctrl-V : Yapıştırır
Ctrl-N : Yeni belge (sekme) açar 
Ctrl-H : Yeni sekme
Ctrl-F : Bul 
Ctrl-H : Değiştir
Ctrl-J : indirilenler (google’da)
Shift-f10 : Farenin Sağ tuşu etkisinde
Enter : Alt satıra geçer 
Alt-f4 : kapat
Shift-f10 : sağ tuş
Shift-del : Kalıcı olarak siler 
Home : Satır başı 
End : satır sonu 
Pagedown : sayfanın en altına gider
Pageup : Sayfanın en üstüne gider 
Scroll lock : Kaydırma çubuğunu kitler 



Yazının büyüklüğüne punto denir.
